# Masquerade Site Template Provenance

Per the project spec, every masquerade-site template must record its source,
license evidence, and the reasoning for inclusion before being bundled.

## default/

- **Source:** Authored from scratch for this project by the maintainers.
- **License:** Dedicated to the public domain under
  [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/).
  No third-party assets, fonts, images, or stylesheets are included; the page
  uses only system fonts and inline CSS.
- **Reason for inclusion:** A minimal, self-contained fallback landing page that
  carries no external dependencies and no licensing obligations, suitable as a
  default masquerade site.

Any future template sourced from a third party must be CC0 / Public Domain only,
and its upstream license file must be stored alongside it in this directory.
